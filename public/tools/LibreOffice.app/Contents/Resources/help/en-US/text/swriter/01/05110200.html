Optimal Row Height

Set row height for selected table rows so that each row has the same height as the row with the tallest content.

To access this command...

Choose Table - Size - Optimal Row Height.

Choose Format - Table - Optimal Row Height.

Select rows in table, right-click, choose Size - Optimal Row Height.

Open Optimize Size dropdown menu from Table bar, click

Open Optimize Size dropdown menu from Table bar, click

Optimal Row Height


Row height can increase with this option, with the table always growing downward. The total table height is never reduced by this option.

This option is similar to first minimizing row height for selected rows by using Minimal Row Height and then distributing those rows by using Distribute Rows Evenly, except that additional height is added to each row if necessary to prevent the total table height from reducing.
