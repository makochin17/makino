Insert Cells

Opens the Insert Cells dialog, in which you can insert new cells according to the options that you specify. You can delete cells by choosing Edit - Delete Cells.

To access this command...

Choose Sheet - Insert Cells.

Choose View - Toolbars and select the Insert Cells toolbar:

Insert Cells


Selection

This area contains the options available for inserting cells into a sheet. The cell quantity and position is defined by selecting a cell range in the sheet beforehand.

Shift cells down

Moves the contents of the selected range downward when cells are inserted.

Insert Cells Down

Shift cells right

Moves the contents of the selected range to the right when cells are inserted.

Insert Cells Right

Entire row

Inserts an entire row. The position of the row is determined by the selection on the sheet. The number of rows inserted depends on how many rows are selected. The contents of the original rows are moved downward.

Insert Rows

Entire column

Inserts an entire column. The number of columns to be inserted is determined by the selected number of columns. The contents of the original columns are shifted to the right.

Insert Columns
