In Front of Object

Changes the stacking order by moving the selected object in front of an object that you specify. The screen location of the selected object does not change.

To access this command...

Choose Shape - Arrange - In Front of Object (LibreOffice Draw only)

Open the context menu of a selected object and choose Arrange - In Front of Object

On the Drawing bar, open the Arrange toolbar and click:

In Front of Object


Select the object(s) that you want to move to the foreground. Right-click and choose Arrange â€“ In Front of Object, and then click an object in your slide.
