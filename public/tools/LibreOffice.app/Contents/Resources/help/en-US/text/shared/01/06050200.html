Ordered

Displays different numbering schemes that you can apply.

To access this command...

Choose View - Styles (Command+TF11) - context menu of a Presentation Style (except Background) - choose New/Modify - Numbering tab.

Choose Format - Bullets and Numbering - Ordered tab.

Choose View - Styles (Command+TF11) - context menu of a List Style - choose New/Modify - Ordered tab.

Choose Styles - Manage Styles - context menu of a List Style - choose New/Modify - Ordered tab.

Selection

Click the numbering that you want to use.
