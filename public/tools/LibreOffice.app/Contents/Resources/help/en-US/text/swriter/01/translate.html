Translate

Translate the selected text or the whole document using DeepL automatic translation.

This feature is experimental and may produce errors or behave unexpectedly. To enable it anyway, choose LibreOffice - PreferencesTools - Options - LibreOffice - Advanced and select Enable experimental features checkbox.


To access this command...

Choose Tools - Translate.


To enable the Translate command, set the server URL and API key in LibreOffice - PreferencesTools - Options - Language Settings - DeepL Server.


Translation works in two ways:

  • Whole document: when there is no selection, the whole document is translated. Each paragraph is sent individually in order to minimize the output timeout for each paragraph.

  • Selection: the selected content is sent to the DeepL server for translation.
