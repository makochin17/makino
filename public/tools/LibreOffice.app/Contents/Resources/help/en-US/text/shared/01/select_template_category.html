Select Category

Use this dialog to set the target category for importing or moving templates between categories. You can also use this dialog to create a category.

To access this command...

Open the Template Manager dialog, then:

  • Select a template, open the context menu, select Move.

  • With All Categories selected, select Manage - Import.


Select from Existing Category

Click to select a category in the list to move or import the template.

or Create a New Category

Enter the name of a new category to move or import the template.
